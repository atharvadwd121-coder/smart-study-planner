# Screenshots Directory

This directory contains screenshots of the Smart Study Planner application.

## Required Screenshots for Evaluation

Please add the following screenshots:

1. **main_menu.png** - Main menu interface showing all options
2. **add_task.png** - Task creation interface
3. **view_tasks.png** - Display of all tasks with priorities
4. **study_session.png** - Study session tracking interface
5. **statistics.png** - Study statistics display
6. **recommendations.png** - AI recommendations screen

## How to Add Screenshots

1. Run the application: `python main.py`
2. Navigate through different menu options
3. Take screenshots using your OS screenshot tool
4. Save them in this directory with descriptive names
5. Commit and push to GitHub

## Screenshot Guidelines

- Use PNG format for better quality
- Ensure text is readable
- Capture the full terminal window
- Include timestamps if relevant
- Show actual data/functionality
