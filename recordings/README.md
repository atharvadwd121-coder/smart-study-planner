# Recordings Directory

This directory is for screen recordings (optional) of the Smart Study Planner application.

## Purpose

Screen recordings can demonstrate:
- Complete workflow from start to finish
- Application features in action
- User interaction patterns
- Real-time functionality

## Recommended Recordings

1. **demo_walkthrough.mp4** - Complete application demo (2-3 minutes)
2. **task_management.mp4** - Creating and managing tasks
3. **study_tracking.mp4** - Starting and tracking study sessions
4. **ai_recommendations.mp4** - Viewing AI-powered recommendations

## Recording Guidelines

- Keep videos short and focused (1-3 minutes each)
- Use screen recording software (OBS Studio, QuickTime, etc.)
- Include audio narration if possible
- Show clear, step-by-step interactions
- Demonstrate key features effectively

## Note

Screen recordings are **optional** but highly recommended for showcasing your project's functionality beyond static screenshots.

*If adding recordings, ensure file sizes are reasonable for GitHub (<25MB per file).*
